Smart Roll Monitoring and Jam Detection System (SRMJDS)

ðŸŽ¯ Problem It Solves

In toilet paper production, roll jams, feed misalignments, or irregular paper tension can cause machine downtime, material wastage, or even damage to cutting and wrapping equipment. Currently, many of these issues are detected manually or too late. The SRMJDS embedded system solves this by providing real-time monitoring and early detection.


ðŸ’¡ Project Summary

The SRMJDS is a compact, low-cost Smart Sensor Module designed using embedded systems to:

  • Monitor paper feed tension in real-time.
  • Detect roll jams or slowdowns early using vibration and optical sensors.
  • Alert the operator with buzzer/LED or integrate into a dashboard or HMI.
  • Optionally log data over time for predictive maintenance.


ðŸ§° Core Components

  • Microcontroller: ESP32 or STM32 (WiFi-capable for data logging)
  • Load cell / strain gauge: For tension measurement
  • IR sensor or rotary encoder: For monitoring roll movement speed
  • Vibration sensor (ADXL335 or SW-420): For abnormal vibration/jam detection
  • OLED Display or LED + buzzer: For local alerts
  • Optional: SD card module or WiFi/Bluetooth to send logs to a PC/app

Impact

  • Reduces unplanned machine downtime
  • Minimizes material waste
  • Increases operator response time and machine efficiency
  • Opens the door to future predictive maintenance or IoT integration in your factory

Future Improvements

  • Add wireless modules (LoRa or WiFi) for a smart factory network
  • Create a dashboard to view real-time data across machines
  • Apply ML models to predict faults before they happen using collected data

Conclusion

The Smart Roll Monitoring and Jam Detection System (SRMJDS) demonstrates the practical application of embedded systems to solve real-world industrial problems. By integrating sensors, actuators, and intelligent data processing, this project reduces downtime, improves efficiency, and lays the foundation for future smart factory innovations.

THE OPERATIONS TO BE FOLLOWED

ðŸ“‹ Project Stages


1. Hardware Design

  • Select and integrate the microcontroller (ESP32 or STM32).
  • Connect load cell/strain gauge for tension measurement.
  • Install IR sensor or rotary encoder for roll speed monitoring.
  • Attach vibration sensor (ADXL335 or SW-420) for jam detection.
  • Set up OLED display, LED, and buzzer for alerts.
  • Optional: Add SD card module or WiFi/Bluetooth for data logging.

2. Firmware Development

  • Write code to read sensor data (analog and digital inputs).
  • Implement algorithms for jam detection and tension monitoring.
  • Program alert logic for buzzer/LED and display updates.
  • Optional: Develop routines for data logging and wireless communication.

3. System Integration & Testing

  • Assemble all hardware components on a test rig or prototype machine.
  • Upload and debug firmware on the microcontroller.
  • Test sensor readings and system response to simulated jams and tension changes.
  • Verify alert mechanisms and data logging functionality.

4. Deployment & Evaluation

  • Install the SRMJDS module on a production machine.
  • Monitor real-world performance and operator feedback.
  • Collect data for predictive maintenance and future improvements.

5. Future Improvements

  • Add wireless modules (LoRa or WiFi) for smart factory networking.
