Mobile Pick and Place Robot with Intelligent Sorting and Counting

üéØ Problem Statement

In packaging and distribution environments, products such as 4-packs, 10-packs, and serviettes must be sorted, counted, and stored efficiently. Manual sorting is slow and error-prone. An automated mobile robot can improve accuracy, speed, and inventory management.

üí° Project Summary

Design and build a mobile pick and place robot that:

  • Moves along a conveyor or workspace.
  • Identifies and distinguishes between 4-packs, 10-packs, and serviettes using sensors and image processing.
  • Picks each item and places it in the correct storage bin.
  • Maintains a real-time count of each group.
  • Stores count data for inventory and reporting.

üß∞ Core Components

  • Mobile Robot Base: Motorized chassis with wheels and navigation sensors (ultrasonic, IR, or lidar).
  • Pick and Place Arm: Servo or stepper motor-driven robotic arm with gripper, designed to reach and stack items up to a height of 2.5 metres.
  • Vision System: Camera module (e.g., OpenMV, Raspberry Pi Camera) for product identification.
  • Microcontroller: ESP32, Arduino, or Raspberry Pi for control and processing.
  • Storage Bins: Separate bins for 4-packs, 10-packs, and serviettes, with vertical stacking capability up to 2.5 metres.
  • Display/Interface: LCD or web dashboard for count monitoring.
  • Power Supply: Rechargeable battery pack.

  • Robot navigates to the pick-up area.

  • Microcontroller updates the count for each group and stores the data.
  • Counts are displayed in real-time and can be exported for inventory management.
